# 확장 프로그램 새로고침 방법

오류가 발생하는 경우, 다음 단계를 따라 확장 프로그램을 완전히 새로고침하세요:

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단 "개발자 모드" 활성화 확인
3. "다독 - 심리 상태 기록" 확장 프로그램 찾기
4. **"새로고침" 버튼 클릭** (또는 확장 프로그램을 제거 후 다시 로드)
5. 페이지를 완전히 새로고침 (Ctrl+Shift+R 또는 Cmd+Shift+R)
6. 콘솔 오류 확인

## 문제가 지속되는 경우

- 확장 프로그램을 완전히 제거한 후 다시 설치
- 브라우저 캐시 완전 삭제
- 시크릿 모드에서 테스트
